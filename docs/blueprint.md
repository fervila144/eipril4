# **App Name**: Catalogo Minimal

## Core Features:

- Visualización de Productos: Muestra todos los productos subidos en una cuadrícula limpia y organizada con información esencial como nombre, imagen y precio.
- Formulario de Carga de Productos: Permite a los usuarios añadir nuevos productos proporcionando detalles como nombre, descripción, precio y la opción de subir una imagen del producto.
- Edición de Productos: Permite modificar fácilmente los detalles de los productos existentes en el catálogo.
- Eliminación de Productos: Funcionalidad para borrar productos específicos del catálogo.
- Asistente de Descripción con IA: Una herramienta de IA que ayuda a generar descripciones concisas y atractivas para los productos, basada en las palabras clave y detalles básicos proporcionados por el usuario, disponible en español.

## Style Guidelines:

- Paleta de colores fría y profesional para una estética limpia y ordenada. El color primario es un azul grisáceo profundo (#3E5074), que inspira confianza y sobriedad. El color de fondo es un blanco azulado muy claro (#F0F2F4), proporcionando una base neutra que destaca el contenido.
- El color de acento es un azul cielo vibrante (#47BAD1), utilizado para resaltar elementos interactivos y llamadas a la acción, manteniendo un equilibrio con el esquema minimalista general.
- La tipografía recomendada es 'Inter' (sans-serif), elegida por su legibilidad y estilo moderno y neutro, ideal para titulares y cuerpo de texto en una interfaz minimalista en español.
- Se utilizarán iconos minimalistas basados en líneas simples y de contorno, manteniendo la coherencia visual con el diseño general y facilitando la comprensión en la interfaz en español.
- Se empleará un diseño basado en cuadrículas para la visualización de productos, con amplio espacio en blanco alrededor de los elementos para mejorar la claridad y facilitar la navegación, promoviendo una experiencia de usuario sin distracciones.
- Animaciones sutiles y discretas, como transiciones suaves al cargar productos o efectos de hover minimalistas, para proporcionar retroalimentación al usuario sin saturar la experiencia visual.
